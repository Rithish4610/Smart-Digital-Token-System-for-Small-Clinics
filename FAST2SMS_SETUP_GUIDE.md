# Fast2SMS Setup Guide

Follow these steps to get your API Key for sending SMS.

## 1. Login to Fast2SMS
- Go to [https://www.fast2sms.com/](https://www.fast2sms.com/).
- Click **Login** / **Sign Up**.
- Verify your mobile number if asked.

## 2. Go to Dev API
- Once logged in, look at the sidebar on the left.
- Click on the **Dev API** option.

## 3. Copy Your API Key
- You will see a box labeled **API key** or **Authorization Key**.
- Click the **Copy** button next to it.
- **Save this key**, you will need to give it to me or paste it into your configuration file.
